## 1. Install the Patly library

```bash
pip install patly
```

## 2. Register your wallet

```bash
curl -X POST https://api.patly.online/register \
  -H "Content-Type: application/json" \
  -d '{"wallet": "0xYOUR_POLYMARKET_WALLET"}'
```

Save the `api_key` from the response.

## 3. Enable the Patly module (one-time)

This grants Patly permission to redeem on your Safe. Your private key is used locally — it never leaves your machine.

```python
from patly import Patly

p = Patly(api_key="YOUR_API_KEY", pk="YOUR_PRIVATE_KEY")
p.enable_module()
# ✅ Module enabled: 0x...
```

<Note>
This sends one transaction to your Polymarket Safe calling `enableModule(patly_wallet)`. 
It requires a tiny amount of POL (~$0.001) for gas — or use the Patly dashboard to enable via Polygonscan.
</Note>

## 4. Add Patly to your bot

```python
# In your bot's startup
from patly import Patly

patly = Patly(api_key="YOUR_API_KEY", pk="YOUR_PRIVATE_KEY")

# After each winning trade:
patly.redeem("btc-updown-15m-1773960300", "YES")
# ✅ Redeemed: 0x4398...  $5.00 USDC back in your wallet
```

## Drop-in replacement

If you're already using a `redeem_if_needed()` function in your bot:

```python
import patly

# At startup
patly.init(api_key="YOUR_API_KEY", pk="YOUR_PRIVATE_KEY")

# This replaces your existing redeem call — same signature
patly.redeem_if_needed(position["slug"], won, won_side=position["side"])
```

## Check status

```python
print(patly.status())
# {"module_enabled": true, "wallet": "0x...", "ready": true}

print(patly.positions())
# [{"slug": "btc-updown-...", "status": "redeemed", "tx_hash": "0x..."}, ...]
```
