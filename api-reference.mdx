http://patly.duckdns.org

## POST /register

Register your Polymarket wallet and receive an API key. **Free.**

### Body

| Field | Type | Description |
|---|---|---|
| `wallet` | string | Your Polymarket wallet address (`0x...`) |

### Response

```json
{
  "api_key": "vRkrRS5q...",
  "wallet":  "0xb1cb60...",
  "message": "Save your API key — it won't be shown again."
}
```

**Errors:** `400` invalid address · `409` already registered

## POST /redeem

Relay a signed redeem transaction. **Requires auth.**

The Patly library builds and signs the transaction automatically. The $0.01 fee is included in the transaction itself — no separate payment step.

### Headers

| Header | Description |
|---|---|
| `x-api-key` | Your Patly API key |

### Body

| Field | Type | Description |
|---|---|---|
| `signed_tx` | object | Safe multisend transaction signed locally by the Patly client |
| `slug` | string | Market slug e.g. `btc-updown-15m-1234567890` |
| `won_side` | string | `"YES"` or `"NO"` |

<Note>
`signed_tx` is a Safe multisend containing two calls: `redeemPositions()` and `USDC.transfer($0.01 → Patly)`. It is built and signed by the Patly library on the user's machine. The private key is never transmitted — only the cryptographic signature.
</Note>

### Response

```json
{
  "status":      "redeemed",
  "tx_hash":     "0x4398...",
  "slug":        "btc-updown-15m-1234567890",
  "won_side":    "YES",
  "polygonscan": "https://polygonscan.com/tx/0x4398..."
}
```

**Errors:** `401` invalid key · `400` bad request · `502` relay failed

## GET /redeems

Your full redeem history. **Requires auth.**

### Query Parameters

| Param | Default | Description |
|---|---|---|
| `limit` | 50 | Max records to return |

### Response

```json
[
  {
    "slug":       "btc-updown-15m-1234567890",
    "won_side":   "YES",
    "tx_hash":    "0x4398...",
    "status":     "redeemed",
    "created_at": "2026-03-20T11:10:07"
  }
]
```

## POST /recover

Recover a lost API key by proving wallet ownership via a MetaMask signature.

### Body

| Field | Type | Description |
|---|---|---|
| `wallet` | string | Your wallet address |
| `message` | string | The message you signed |
| `signature` | string | The eth_sign signature |

### Response

```json
{ "api_key": "...", "wallet": "0x..." }
```

## DELETE /account

Permanently delete your account and all records. **Requires auth.**

```bash
curl -X DELETE http://patly.duckdns.org/account \
  -H "x-api-key: YOUR_API_KEY"
```

```json
{ "status": "deleted" }
```
