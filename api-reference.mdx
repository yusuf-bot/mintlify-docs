http://patly.duckdns.org

## POST /register

Register your Polymarket wallet and receive an API key. **Free.**

### Body

| Field | Type | Description |
|---|---|---|
| `wallet` | string | Your Polymarket wallet address (`0x...`) |

### Response

```json
{
  "api_key": "vRkrRS5q...",
  "wallet":  "0xb1cb60...",
  "message": "Save your API key — it won't be shown again."
}
```

**Errors:** `400` invalid address · `409` already registered

## GET /status

Your account status and redeem stats. **Requires auth.**

### Headers

| Header | Description |
|---|---|
| `x-api-key` | Your Patly API key |

### Response

```json
{
  "wallet":   "0xb1cb60...",
  "redeemed": 14,
  "pending":  0,
  "failed":   0
}
```

## POST /redeem

Relay a signed redeem transaction. **Costs $0.01 USDC via x402.**

The Patly library handles signing and payment automatically. If calling manually, see the x402 protocol docs.

### Headers

| Header | Description |
|---|---|
| `x-api-key` | Your Patly API key |
| `X-PAYMENT` | x402 payment signature (handled by Patly library) |

### Body

| Field | Type | Description |
|---|---|---|
| `signed_tx` | object | Safe transaction signed locally by user's PK |
| `slug` | string | Market slug e.g. `btc-updown-15m-1234567890` |
| `won_side` | string | `"YES"` or `"NO"` |

<Note>
The `signed_tx` is built and signed by the Patly library on the user's machine. The private key is never transmitted — only the cryptographic signature is included.
</Note>

### Response

```json
{
  "status":      "redeemed",
  "tx_hash":     "0x4398...",
  "slug":        "btc-updown-15m-1234567890",
  "won_side":    "YES",
  "polygonscan": "https://polygonscan.com/tx/0x4398..."
}
```

**Errors:** `402` payment required · `401` invalid key · `400` bad request · `502` relay failed

## GET /redeems

Your full redeem history. **Requires auth.**

### Query Parameters

| Param | Default | Description |
|---|---|---|
| `limit` | 50 | Max records to return |

### Response

```json
[
  {
    "slug":       "btc-updown-15m-1234567890",
    "won_side":   "YES",
    "tx_hash":    "0x4398...",
    "status":     "redeemed",
    "created_at": "2026-03-20T11:10:07"
  }
]
```

## DELETE /account

Permanently delete your account and all records. **Requires auth.**

```bash
curl -X DELETE http://patly.duckdns.org/account \
  -H "x-api-key: YOUR_API_KEY"
```

```json
{ "status": "deleted" }
```
