## Register

Register your Polymarket Safe wallet address. Returns an API key used to authenticate all future requests. Free — no payment required.

### Body

<ParamField body="wallet" type="string" required>
  Your Polymarket wallet address (your Safe address, e.g. `0xb1Cb60...`)
</ParamField>

### Response

<ResponseField name="api_key" type="string">
  Your API key. Store this securely — it authenticates all requests.
</ResponseField>

<ResponseField name="wallet" type="string">
  Your registered wallet address.
</ResponseField>

<ResponseField name="next" type="string">
  Next step — enable the Patly module.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.patly.online/register \
  -H "Content-Type: application/json" \
  -d '{"wallet": "0xYOUR_WALLET_ADDRESS"}'
```

```python Python
from patly import Patly
# Registration happens automatically on first use
# or via the dashboard at patly.online
```
</RequestExample>

<ResponseExample>
```json 200
{
  "api_key": "S-s4H6krlSRP62MLy...",
  "wallet": "0xb1cb60c750eeea1830773825fe85c419f44c544d",
  "next": "Enable module: call enableModule(0xPATLY_WALLET) on your Polymarket Safe"
}
```

```json 409
{
  "detail": "Wallet already registered"
}
```
</ResponseExample>

# Check Status

<api>GET https://api.patly.online/status</api>

Check whether the Patly module is enabled on your Safe.

### Headers

<ParamField header="x-api-key" type="string" required>
  Your Patly API key.
</ParamField>

<RequestExample>
```bash cURL
curl https://api.patly.online/status \
  -H "x-api-key: YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```json 200 — Module enabled
{
  "wallet": "0xb1cb60...",
  "module_enabled": true,

```
</ResponseExample>

# Redeem

<api>POST https://api.patly.online/redeem</api>

Redeem a winning Polymarket position. Costs **$0.05 USDC** per call via x402 — handled automatically by the Patly Python library.

<Note>
This endpoint returns **HTTP 402** on the first call with payment instructions. The Patly library handles this automatically. If calling manually, see [x402 buyer docs](https://x402.org/docs/buyers).
</Note>

### Headers

<ParamField header="x-api-key" type="string" required>
  Your Patly API key.
</ParamField>

<ParamField header="X-PAYMENT" type="string">
  x402 payment signature (handled automatically by the Patly library).
</ParamField>

### Body

<ParamField body="slug" type="string" required>
  The market slug, e.g. `btc-updown-15m-1773960300`
</ParamField>

<ParamField body="won_side" type="string" required>
  `"YES"` or `"NO"` — which side you won.
</ParamField>

<RequestExample>
```python Patly Library (recommended)
from patly import Patly

p = Patly(api_key="YOUR_KEY", pk="YOUR_PK")
result = p.redeem("btc-updown-15m-1773960300", "YES")
print(result["tx_hash"])
```

```bash cURL (manual x402)
# First call returns 402 with payment instructions
curl -X POST https://api.patly.online/redeem \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -H "X-PAYMENT: BASE64_PAYMENT_PAYLOAD" \
  -d '{"slug": "btc-updown-15m-1773960300", "won_side": "YES"}'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "status": "redeemed",
  "tx_hash": "0x4398319d0d0cf3761fec38057cbd6830451fb46ff4ebc42290cb6a4ba2d08cd6",
  "slug": "btc-updown-15m-1773960300",
  "won_side": "YES",
  "polygonscan": "https://polygonscan.com/tx/0x4398..."
}
```

```json 402
{
  "x402Version": 1,
  "accepts": [{
    "scheme": "exact",
    "network": "eip155:137",
    "payTo": "0xSERVICE_WALLET",
    "amount": "50000",
    "asset": "0x2791Bca1f2de4661ED88A30C99A7a9449Aa84174"
  }],
  "error": "Payment required"
}
```

```json 403
{
  "detail": "Module not enabled. Call enableModule(0x...) on your Safe first."
}
```
</ResponseExample>

# List Positions

<api>GET https://api.patly.online/positions</api>

List all registered positions and their redemption status.

### Headers

<ParamField header="x-api-key" type="string" required>
  Your Patly API key.
</ParamField>

### Query Parameters

<ParamField query="limit" type="integer" default="50">
  Maximum number of positions to return.
</ParamField>

<RequestExample>
```bash cURL
curl "https://api.patly.online/positions?limit=20" \
  -H "x-api-key: YOUR_API_KEY"
```

```python Python
positions = patly.positions()
for p in positions:
    print(p["slug"], p["status"])
```
</RequestExample>

# Delete Account

<api>DELETE https://api.patly.online/account</api>

Permanently delete your Patly account and all position records. The module will remain enabled on your Safe — disable it manually if desired.

### Headers

<ParamField header="x-api-key" type="string" required>
  Your Patly API key.
</ParamField>

<RequestExample>
```bash cURL
curl -X DELETE https://api.patly.online/account \
  -H "x-api-key: YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```json 200
{
  "status": "deleted"
}
```
</ResponseExample>
