# :Patly

**Patly** is a Redeem-as-a-Service API for Polymarket. It automatically converts your winning conditional tokens back to USDC — gaslessly, with no code changes to your bot.

## The Problem

Polymarket positions don't auto-redeem. After a market resolves, your winning tokens sit idle until you manually call `redeemPositions()` on-chain. Bots that don't handle this run out of trading capital.

## The Solution

Patly uses Gnosis Safe's **Module system**. You grant Patly permission once — after that, Patly can redeem on your behalf anytime, without ever touching your private key again.

```
You (once):    enableModule(patly_wallet) → on your Polymarket Safe
Your bot:      POST /redeem { slug, won_side } → Patly redeems → USDC in your wallet
```

## Key Properties

- **No private key storage** — Patly never sees your PK after setup
- **Gasless** — Patly uses Polymarket's official relayer (Builder program)
- **$0.05 per redeem** — paid via x402 (automatic in the Patly library)
- **Instant** — redeems execute the moment you call the API
